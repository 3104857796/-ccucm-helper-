# 校园AI助手 — 部署指南

预计耗时：15分钟

---

## 第一步：获取 DeepSeek API Key（5分钟）

1. 打开 **https://platform.deepseek.com** → 点右上角「登录/注册」
2. 用手机号注册，新用户送 500万 tokens 免费额度（够用很久）
3. 登录后点左侧菜单「**API Keys**」→ 点「**创建新的 API Key**」
4. 起个名字"校园助手"，复制生成的 key（以 `sk-` 开头）
5. 打开 `config.js`，找到 `deepseekApiKey: 'sk-你的key填这里'`，把 key 粘贴进去

> DeepSeek 价格：100万 tokens ≈ 1块钱。一条校园问答约 1000 tokens，等于一分钱。免费额度用完充10块够用一个学期。

---

## 第二步：填你的群信息（2分钟）

打开 `config.js`，修改群号：

```javascript
groups: {
    default: { name: '新生总群', id: '把你的群号填这里', tip: '...' },
    study:   { name: '学业交流群', id: '把你的群号填这里', tip: '...' },
    // ... 以此类推
},
```

如果暂时没有这么多群，可以先全部填同一个群。

---

## 第三步：部署到 GitHub Pages（免费，8分钟）

### 3.1 注册 GitHub
- 打开 https://github.com → 点右上角 Sign up
- 用邮箱注册（不需要科学上网）

### 3.2 创建仓库
- 点右上角 **+** → **New repository**
- Repository name 填：`ccucm-helper`（或任意名字）
- 选 **Public**
- 点 **Create repository**

### 3.3 上传文件
创建仓库后，页面会有一个「uploading an existing file」链接，点它。
然后把这三个文件拖进去：
- `index.html`
- `config.js`
- （知识库已内置在 config.js 中，不需要单独上传）

拖完点「**Commit changes**」按钮。

### 3.4 开启 GitHub Pages
- 进入仓库 → 点顶部「**Settings**」
- 左侧菜单点「**Pages**」
- Branch 选 `main`，点 **Save**
- 等 1-2 分钟，页面刷新后会出现一个网址：
  `https://你的用户名.github.io/ccucm-helper/`
- 点进去就能看到你的校园助手了！

## 第五步：换学校怎么办？

只需改 `config.js` 里的三样东西：
1. 学校名称
2. 群信息
3. 知识库文字（从 `新生问答知识库-长春中医药大学.txt` 替换）

改完重新上传到 GitHub，**自动生效**，网址不变。

## 可选：加密码保护

如果只想让群里的人用，在 `config.js` 里设置：
```javascript
password: '2024ccucm',
```
然后单独告诉新生验证码。外人打开网页会看到密码输入页。

## 遇到问题？

| 问题 | 解决 |
|------|------|
| 网页打不开 | GitHub Pages 需要等1-2分钟生效，刷新试试 |
| 发送消息没反应 | 检查 `config.js` 里 API Key 填对没有 |
| 回答很慢 | DeepSeek 免费版偶尔排队，一般3-10秒 |
| API 报错 | 去 platform.deepseek.com 看余额和额度 |
